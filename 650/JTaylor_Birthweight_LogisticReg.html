Low Birth Weight Logistic Regression Analysis - James Taylor¶

This notebook uses data on births and if the child was underweight or not. It will explore the dataset, make some visualizations and create a logistic regression.¶

The variables and their descriptions are provided below. There are 9 of them in total.

VariableDescription
IDIdentification Code
LOWLow birth weight (0: >= 2500, 1: + 2500g) - target variable
AGEMothers age in years
RACERace (1: White, 2: Black, 3: Other)
SMOKESmoking status during pregnancy (0: No, 1: Yes)
PTLHistory of premature labor (0: None, 1: One, 2: Two, etc)
HTHistory of hypertension (0: No, 1: Yes)
UIPresence of Uterine irritability (0: No, 1: Yes)
FTVNumber of physician visits during the first trimester (0: None, 1: One, 2: Two, etc)

Transformations¶

It doesn't make sense for us to include the RACE variable as it was provided. It has numerical values for different races and the logistic regression will treat the variable as any other numerical value. Instead we will ctreate two dummy variables for 'White' and 'Black' races. This will let us inspect the estimate of each race indivdualy. Creating these variables will make the given RACE variable not needed.

In [16]:

The two columns furthest to the right are the newly populated dummy variables. Obviously when someone is white (RACE = 1) the WHITEbool dummy varible should be True and BLACKbool dummy should be False. The two dummy should never both be True, or both = 1 becuase someone cannot be black and white given the data we originaly recived. However, they can both be False becuase there is the 'Other' race category, which there is no dummy variable for. There is no varible for 'Other' becuase it would cause a perfect-multicolinearity problem in the regression.
